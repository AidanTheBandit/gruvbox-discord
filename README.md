# gruvbox-discord

A dark, retro groove themed Discord theme based on the [Gruvbox color scheme](https://github.com/morhetz/gruvbox).

This theme is a customized version of [midnight-discord](https://github.com/refact0r/midnight-discord) with Gruvbox colors applied throughout.

## color scheme

This theme uses the Gruvbox dark color palette:
- Background: `#282828` (dark0)
- Foreground: `#ebdbb2` (light1)
- Accent: `#689d6a` (neutral aqua)
- Status colors: 
  - Online: `#b8bb26` (bright green)
  - DND: `#fb4934` (bright red)
  - Idle: `#fabd2f` (bright yellow)
  - Streaming: `#d3869b` (bright purple)

## install

### option 1: ready-to-use build (recommended for most users)

1. download the ready build file, [`midnight.css`](https://github.com/AidanTheBandit/gruvbox-discord/blob/master/build/midnight.css). (there should be a download button at the top right of the page)
2. rename it to `gruvbox.css` or any name you prefer
3. drag the file into your theme folder. (there should be a button to open the theme folder in theme settings)

this file contains the complete theme with all gruvbox colors pre-applied and ready to use!

### option 2: theme file (for advanced customization)

1. download the theme file, [`midnight.theme.css`](https://github.com/AidanTheBandit/gruvbox-discord/blob/master/themes/midnight.theme.css). (there should be a download button at the top right of the page)
2. drag the file into your theme folder. (there should be a button to open the theme folder in theme settings)
3. (optional) customize the theme by editing the options in `midnight.theme.css`.

### option 3: install through link (for github pages hosting)

add `https://aidanthebandit.github.io/gruvbox-discord/build/midnight.css` to your theme import links. you will need to copy the theme variables to your quickcss in order to customize the theme.

note: this requires github pages to be enabled for this repository.
